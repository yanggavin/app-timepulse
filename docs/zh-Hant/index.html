時脈支援頁面。

此頁面提供訂閱、iCloud 同步、資料控制與一般使用者支援說明。

支援信箱：support@timepulse-app.newtree.tech 網站：https://timepulse-app.newtree.tech/

常見問題

  • 訂閱與購買： 時脈 Pro 透過 Apple 完成扣款，你可以在 Apple ID 的訂閱設定中管理或取消。
  • 恢復購買： 在付費牆點擊「恢復購買」，或在應用設定中使用恢復流程。
  • iCloud 同步： Pro 使用者可在設定中啟用 iCloud 同步，資料透過同一 Apple ID 下的 CloudKit 同步。
  • 清除資料： 可在時脈設定中清除本機或同步資料；若已開啟同步，請先閱讀確認文案。
  • 無需帳號： 使用時脈不需要另外註冊時脈帳號。
