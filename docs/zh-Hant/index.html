時脈技術支援

時脈（TimePulse）習慣追蹤、翻頁時鐘、情緒紀錄與可選 iCloud 同步的使用說明。

聯絡我們

如有問題或意見，請寄信至 support@timepulse-app.newtree.tech。

常見主題

  • iCloud 同步（時脈 Pro） — 請先在系統設定中為裝置開啟 iCloud，再在時脈內啟用同步。資料透過你的 Apple ID 與 CloudKit 同步；我們不會在自己的伺服器上代管你的習慣或情緒內容。
  • 訂閱 — 款項由 Apple 處理。可在 App Store 中前往 Apple ID → 訂閱項目管理或取消，或在「設定」→ [你的名字] →「訂閱項目」操作。
  • 回復購買 — 開啟時脈 → 設定，使用 App 內提供的訂閱／回復項目（依版本可能為「顧客中心」或系統回復流程）。
  • 本機資料 — 可在時脈設定中清除資料。若已開啟 iCloud 同步，清除可能影響同一 iCloud 帳號下的其他裝置，請詳閱 App 內確認提示。
