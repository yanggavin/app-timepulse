Common topics

  • Subscriptions and purchases: TimePulse Pro purchases are processed by Apple. Manage or cancel them in your Apple ID subscriptions settings.
  • Restore purchases: Open the paywall and tap Restore Purchases, or use the restore flow in Settings.
  • iCloud sync: Pro users can enable iCloud sync in Settings. Sync uses CloudKit with the same Apple ID signed into the device.
  • Clear data: Local data can be removed inside TimePulse settings. If iCloud sync is enabled, read the confirmation message carefully before clearing synced data.
  • No account required: The app works without creating a TimePulse account.
