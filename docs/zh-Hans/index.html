时脉支持页面。

此页面提供订阅、iCloud 同步、数据控制和一般用户支持说明。

支持邮箱：support@timepulse-app.newtree.tech 网站：https://timepulse-app.newtree.tech/

常见问题

  • 订阅与购买： 时脉 Pro 通过 Apple 完成扣费，你可以在 Apple ID 的订阅设置中管理或取消。
  • 恢复购买： 在付费墙点击“恢复购买”，或在应用设置里使用恢复流程。
  • iCloud 同步： Pro 用户可在设置中启用 iCloud 同步，数据通过同一 Apple ID 下的 CloudKit 同步。
  • 清除数据： 可在时脉设置中清除本地或同步数据；如果已开启同步，请先阅读确认文案。
  • 无需账号： 使用时脉不需要单独注册时脉账号。
