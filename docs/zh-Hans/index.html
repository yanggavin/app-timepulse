时脉技术支持

时脉（TimePulse）习惯追踪、翻页时钟、情绪记录与可选 iCloud 同步的使用帮助。

联系我们

如有问题或建议，请发送邮件至 support@timepulse-app.newtree.tech。

常见问题

  • iCloud 同步（时脉 Pro） — 请先在系统设置中为设备开启 iCloud，再在时脉内打开同步。数据通过你的 Apple ID 与 CloudKit 同步；我们不会在自己的服务器上托管你的习惯或情绪内容。
  • 订阅 — 付款由 Apple 处理。可在 App Store 中进入 Apple ID → 订阅进行管理或取消，或在 设置 → [你的名字] → 订阅 中操作。
  • 恢复购买 — 打开时脉 → 设置，使用应用内提供的订阅 / 恢复入口（具体文案因版本可能为「客户中心」或系统恢复流程）。
  • 本机数据 — 可在时脉设置中清除数据。若已开启 iCloud 同步，清除可能影响同一 iCloud 账号下的其他设备，请仔细阅读应用内确认提示。
